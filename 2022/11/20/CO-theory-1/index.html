定点数的表示

无符号数和有符号数

  • 无符号数
    没有符号位，8位数表示范围0 ~ 2^8^-1即0 ~ 255
  • 有符号数
    有一个符号位

定点表示

小数点位置不变

  • 定点小数
    纯小数，第一位是符号位,符号位后全是小数位
    • x~0~ = 0 ，最大值 1-2^-n^
    • x~0~ = 0 ，最小值 -(1-2^-n^)
  • 定点整数
    纯整数，第一位是符号位，符号位后全是整数位
    • x~0~ = 0 ，最大值 2^-n^ -1
    • x~0~ = 0 ，最小值 -(2^-n^ -1)

原码、补码、反码（有符号位）

  • 原码
    第一位符号位，其余位为绝对值

    • 小数原码
      [0.1]~原~ = 0.1000000
      [-0.1]~原~ = 1.1000000

    • 整数原码
      [1]~原~ = 00000001
      [-1]~原~ = 10000001

      不同符号数加法需要先比较绝对值大小，再相减

  • 补码

    • 纯小数补码

    • 整数补码
      [正]~补~ = 0（符号位），原值
      [负]~补~ = 1（符号位），原值取反加一

      整数补码的零唯一，且多一个-2^n^

  • 反码
    零不唯一，[+0]~反~ = 0.0000,[-0]~反~ = 1.1111
  • 移码

定点数的计算

定点数的移位

  • 算术移位
    有符号位！
    符号位不变，无论正负原码都移位后补0，负数反补码会有差异。
    注意计算机的负数是由补码表示的，其实就是补充最高位。
  • 逻辑移位
    无符号位
    移位都添0
  • 循环移位

定点数的加减法

  • 原码的加减法
    • 加法
      先判符号位，绝对值加减，再补符号位
      溢出位丢掉
    • 减法
      减数符号取反加原码
  • 补码的加减法
    计算机普遍采用补码加减
    • 补码加减补码等于补码
    • 符号位参与运算，进位丢掉
